api: DELETE /documents/{id}

### Authorizations

<ParameterField name="Authorization" type="string" required>
  Bearer authentication header of the form `Bearer <token>`.
</ParameterField>

### Path Parameters

<ParameterField name="id" type="integer (int64)" required>
  ID of document to delete
</ParameterField>
